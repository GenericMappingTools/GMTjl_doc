GDAL Utility Functions

Functions for working with geospatial data using GDAL/OGR capabilities.

Core GDAL Tools

  • gdaltranslate - Convert raster data between different formats
  • gdalwarp - Image reprojection and warping
  • gdaldem - Tools to analyze and visualize DEMs
  • ogr2ogr - Convert vector data between file formats

Geometric Operations

  • arccircle - Create circular arcs
  • arcellipse - Create elliptical arcs
  • boundary - Compute geometry boundary
  • buffer - Create buffer around geometry
  • buffergeo - Geographic buffer operations
  • centroid - Compute centroid of geometry
  • circgeo - Geographic circles
  • concavehull - Compute concave hull
  • convexhull - Compute convex hull
  • delaunay - Delaunay triangulation
  • envelope - Compute bounding envelope
  • envelope3d - Compute 3D bounding envelope
  • polygonize - Convert raster to polygons
  • polyunion - Union of polygons
  • simplify - Simplify geometry

Spatial Relationships

  • contains - Test if geometry contains another
  • crosses - Test if geometries cross
  • disjoint - Test if geometries are disjoint
  • equals - Test if geometries are equal
  • intersects - Test if geometries intersect
  • overlaps - Test if geometries overlap
  • touches - Test if geometries touch
  • within - Test if geometry is within another

Distance & Area Calculations

  • distance - Compute distance between geometries
  • geodesicarea - Compute geodesic area
  • geomarea - Compute geometry area
  • geomlength - Compute geometry length

Geodetic Operations

  • geod - Geodesic calculations
  • geodesic - Geodesic line operations
  • invgeod - Inverse geodesic calculations
  • loxodrome - Loxodrome (rhumb line) calculations
  • loxodrome_direct - Direct loxodrome problem
  • loxodrome_inverse - Inverse loxodrome problem

Projection & Coordinate Functions

  • epsg2proj - Convert EPSG code to PROJ string
  • epsg2wkt - Convert EPSG code to WKT
  • lonlat2xy - Convert lon/lat to projected coordinates
  • proj2wkt - Convert PROJ string to WKT
  • setproj! - Set projection for data
  • wkt2proj - Convert WKT to PROJ string
  • xy2lonlat - Convert projected coordinates to lon/lat

Set Operations

  • difference - Compute difference of geometries
  • intersection - Compute intersection of geometries
  • symdifference - Compute symmetric difference

Data Processing

  • dither - Apply dithering to image
  • fillnodata - Fill nodata values in raster

GMT Integration

  • gd2gmt - Convert GDAL dataset to GMT object
  • gdalshade - Create shaded relief
  • gdalread - Read geospatial data with GDAL
  • gdalwrite - Write geospatial data with GDAL
  • gmt2gd - Convert GMT object to GDAL dataset

Miscellaneous

  • pointalongline - Get point at distance along line
